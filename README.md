# Zatowarowanie – Executive AI Suite (v6.1 Branding)

## Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Branding
- Podmień logo w UI (sidebar) lub wgraj do `assets/logo_biadvice.png`.
- Edytuj paletę w `assets/branding.json` (kolory, colorway Plotly).
- Wszystkie wykresy używają motywu z palety (dark business).

## Eksporty
- PDF/PPTX z brandingiem (przycisk w apce).
- CSV rekomendacji.
